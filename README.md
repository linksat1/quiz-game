# 🎯 나만의 퀴즈 게임 (우주/과학 상식)

터미널에서 동작하는 Python 콘솔 퀴즈 게임 프로젝트입니다.
Python 기초 문법, 클래스(객체 지향), JSON 파일 입출력, Git 워크플로우를
직접 익히기 위해 처음부터 끝까지 직접 구현했습니다.

---

## 📖 프로젝트 개요

- **언어 / 환경**: Python 3.10+ (외부 라이브러리 없이 표준 라이브러리만 사용)
- **실행 형태**: 터미널 콘솔 프로그램
- **핵심 기능**: 퀴즈 풀기 / 퀴즈 추가 / 퀴즈 목록 / 최고 점수 확인
- **데이터 영속성**: 프로그램을 종료하고 다시 실행해도 추가한 퀴즈와
  최고 점수가 유지됩니다 (`state.json`).

---

## 🌌 퀴즈 주제 선정 이유 — "우주/과학 상식"

우주와 과학 상식은 아래와 같은 이유로 학습용 퀴즈 주제로 적합하다고 판단했습니다.

1. **정답이 명확하다** — 과학적 사실 기반이라 퀴즈 정답에 논란이 적고,
   정오 판정 로직을 단순하게 구현하기에 적합합니다.
2. **난이도 분포가 넓다** — "태양계에서 가장 큰 행성"처럼 누구나 아는 문제부터,
   "DNA 이중 나선 구조를 발표한 과학자"처럼 깊이 있는 문제까지
   다양한 난이도를 자연스럽게 섞을 수 있습니다.
3. **흥미롭다** — 블랙홀, 달 착륙, 빛의 속도 등은 읽는 것만으로도
   지적 호기심을 자극하므로, 퀴즈를 푸는 재미가 있습니다.

## ▶️ 실행 방법

```bash
# 1. 저장소 클론
git clone <저장소 URL>
cd quiz-game

# 2. 실행 (Python 3.10 이상)
python main.py
```

외부 패키지 설치가 필요 없습니다(`pip install` 불필요).

## 🕹️ 기능 목록

| 메뉴 | 기능 | 설명 |
|------|------|------|
| 1 | 퀴즈 풀기 | 등록된 모든 퀴즈를 순서대로 출제하고 정답 여부와 최종 점수를 보여줍니다. |
| 2 | 퀴즈 추가 | 문제 / 선택지 4개 / 정답 번호를 입력받아 새 퀴즈를 등록하고 파일에 저장합니다. |
| 3 | 퀴즈 목록 | 현재 등록된 퀴즈의 제목(문제) 목록을 번호와 함께 출력합니다. |
| 4 | 점수 확인 | 지금까지의 최고 점수를 확인합니다. (미플레이 시 안내 메시지 출력) |
| 5 | 종료 | 현재 상태를 파일에 저장하고 프로그램을 종료합니다. |

### 예외/입력 처리
- 숫자 입력란에서 **빈 입력 / 공백 / 문자(abc) / 범위 밖 숫자**는
  안내 메시지를 출력하고 재입력을 요청합니다.
- 프로그램 실행 중 `Ctrl+C`(KeyboardInterrupt)나 입력 스트림 종료(`EOFError`)가
  발생해도 비정상 종료하지 않고, 현재 상태를 저장한 뒤 안전하게 종료합니다.
- `state.json` 파일이 없거나 손상되어 있어도 기본 퀴즈 데이터로 복구하여
  프로그램이 항상 정상 실행됩니다.

## 💾 데이터 파일 설명 (`state.json`)

- **경로**: 프로젝트 루트의 `state.json`
- **역할**: 등록된 퀴즈 목록과 최고 점수를 저장/복원한다.
- **인코딩**: UTF-8
- **스키마**:

```json
{
  "quizzes": [
    {
      "question": "태양계에서 가장 큰 행성은 무엇일까요?",
      "choices": ["지구", "토성", "목성", "해왕성"],
      "answer": 3
    }
  ],
  "best_score": 80
}
```

| 필드 | 타입 | 설명 |
|------|------|------|
| `quizzes` | `list[object]` | 등록된 퀴즈 배열 |
| `quizzes[].question` | `string` | 문제 내용 |
| `quizzes[].choices` | `string[4]` | 선택지 4개 |
| `quizzes[].answer` | `int (1~4)` | 정답 번호 |
| `best_score` | `int` | 최고 점수 (100점 만점, 미플레이 시 `-1`) |

> 파일이 없거나 JSON이 손상된 경우, 프로그램은 안내 메시지를 출력한 뒤
> `default_quizzes.py`의 기본 퀴즈로 자동 복구합니다.

## 🏗️ 코드 구조 (클래스 설계)

- **`Quiz`** — 개별 퀴즈 1개를 표현한다.
  - 속성: `question`, `choices`, `answer`
  - 메서드: `display()`, `is_correct()`, `to_dict()`, `from_dict()`
- **`QuizGame`** — 게임 전체를 관리한다.
  - 속성: `quizzes`, `best_score`, `state_path`
  - 메서드: `show_menu()`, `run()`, `play_quiz()`, `add_quiz()`,
    `show_quiz_list()`, `show_best_score()`, `load_state()`, `save_state()`
